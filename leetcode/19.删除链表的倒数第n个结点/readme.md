### 题目
给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

进阶：你能尝试使用一趟扫描实现吗？

示例1：
```
输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]
```

示例 2：
```
输入：head = [1], n = 1
输出：[]
```
示例 3：

```
输入：head = [1,2], n = 1
输出：[1]
```

### 解法一
刚好在剑指offer中刷到过类似的题目：[链表中倒数第k个结点](../../剑指offer/15.链表中倒数第k个结点/solution.py)

按照剑指offer中的解法，使用双指针可以快速找到第n个结点。定位到了之后，记录其前一个结点，然后：
```python
pre.next = pre.next.next
```
